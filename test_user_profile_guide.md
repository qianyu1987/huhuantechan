## 常见问题排查

### Q1: 用户主页显示"加载失败"
**可能原因**：
1. `VALUE_RANGES_V2` 未定义
2. 云函数调用失败
3. 参数传递错误

**解决方案**：
1. 检查 `user-profile/index.js` 中的常量导入
2. 检查云函数部署和调用
3. 检查 openid 参数传递

### Q2: 代购信息不显示
**可能原因**：
1. 云函数未返回代购字段
2. 用户不是代购者
3. 前端未解析代购字段

**解决方案**：
1. 检查 `userInit` 云函数的 `publicProfile` 方法
2. 检查用户数据库中的代购字段
3. 检查前端代购信息显示逻辑

### Q3: 性别徽章不显示
**可能原因**：
1. 性别字段值为空或无效
2. CSS 样式问题
3. WXML 条件渲染问题

**解决方案**：
1. 检查用户性别字段值（0,1,2）
2. 检查 CSS 样式是否正确加载
3. 检查 WXML 条件渲染逻辑

### Q4: 等级计算错误
**可能原因**：
1. 信用分数据错误
2. 等级计算逻辑错误
3. 边界值处理错误

**解决方案**：
1. 检查用户信用分数据
2. 验证等级计算逻辑
3. 测试边界值情况
