# 公众号 IP 白名单配置指南

## 1. 获取云函数出口 IP

云函数的执行环境 IP 不是固定的，但可以通过以下方式获取：

### 方法A：查看腾讯云文档
根据腾讯云文档，云开发（TCB）的出口 IP 段：
- 广州：100.64.0.0/10
- 上海：100.64.0.0/10
- 北京：100.64.0.0/10

### 方法B：通过云函数获取（推荐）
创建一个测试云函数来获取出口 IP：

```javascript
// cloudfunctions/getIP/index.js
exports.main = async () => {
  const https = require('https')
  
  return new Promise((resolve) => {
    https.get('https://api.ipify.org?format=json', (res) => {
      let data = ''
      res.on('data', chunk => data += chunk)
      res.on('end', () => {
        resolve({ success: true, ip: JSON.parse(data).ip })
      })
    }).on('error', (e) => {
      resolve({ success: false, error: e.message })
    })
  })
}
```

调用后可以看到当前云函数的出口 IP。

### 方法C：查看微信官方文档
微信小程序云开发的出口 IP 范围：
- 100.64.0.0/10（腾讯云内网段）

## 2. 配置公众号 IP 白名单

### 步骤：

1. **登录公众号后台**
   - 访问 https://mp.weixin.qq.com
   - 管理员扫码登录

2. **进入开发配置**
   - 左侧菜单：开发 → 基本配置

3. **配置 IP 白名单**
   - 找到 "IP 白名单" 区域
   - 点击 "查看"
   - 点击 "修改"

4. **添加 IP 地址**
   
   由于云函数 IP 不固定，建议添加以下网段：
   ```
   100.64.0.0/10
   ```
   
   或者添加具体 IP（需要多次测试获取）：
   ```
   100.64.x.x
   100.65.x.x
   100.66.x.x
   ...
   ```

5. **保存配置**
   - 需要管理员微信扫码确认

## 3. 替代方案（推荐）

由于云函数 IP 不固定，配置白名单比较麻烦，建议使用以下替代方案：

### 方案A：手动添加文章
在小程序管理后台手动添加公众号文章，不需要配置 IP 白名单。

### 方案B：使用 publish 接口
如果只需要发布文章到公众号，可以使用 `publish` 接口，不需要获取素材列表。

### 方案C：使用第三方服务
使用 RSS 订阅服务（如 Feeddd）获取公众号文章，绕过 IP 限制。

### 方案D：服务器中转
在自己的服务器上部署 API，服务器调用微信接口，小程序调用自己的服务器。

## 4. 当前建议

鉴于配置 IP 白名单的复杂性，建议：

1. **先使用示例文章**：确保文章列表功能正常
2. **手动添加重要文章**：在管理后台手动添加
3. **后续优化**：如果需要自动同步，再考虑配置 IP 白名单或使用服务器中转

## 5. 快速测试

如果您想测试 IP 白名单是否配置正确，可以：

1. 在公众号后台添加 IP：100.64.0.0/10
2. 重新部署云函数
3. 调用同步接口查看是否能获取到文章

如果还是失败，说明可能是其他原因（如公众号未认证、没有图文素材等）。
