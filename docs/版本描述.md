# 小程序版本描述

## 一句话描述
**特产互换**是一款让用户可以在线分享和互换全国各地特色商品的便民服务小程序。

---

## 核心功能描述

### 1. 家乡特产分享
- 用户可以将自己家乡的特色商品拍照上传，与其他用户分享
- 支持文字、图片展示商品信息
- 审核通过后即可展示给其他用户

### 2. 闲置物品互换
- 用户之间可以自由发起互换请求
- 以物换物，环保又省钱
- 双方确认后完成交换

### 3. 信用评价体系
- 每次互换完成后，双方可互相评价
- 信用分公开透明，让交易更放心
- 累计好评可提升信用等级

### 4. 邀请好友
- 邀请新用户注册可获得积分奖励
- 好友完成首次互换，双方均可获得额外积分
