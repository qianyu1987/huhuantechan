P0⏰ 订单超时机制⏱ 4h

确认后 72 小时未发货自动取消订单，释放双方特产库存，发送系统通知。

🗄️ 数据库设计

// orders 表新增字段
{
  confirmTime: Date,        // 确认时间
  shipDeadline: Date,       // 发货截止时间（confirmTime + 72h）
  autoCancelReason: String  // 自动取消原因
}

P1 · 重要功能

体验优化功能，下周完成（25h）

P1⚖️ 纠纷仲裁（人工介入）⏱ 8h

用户可申请纠纷，管理员后台处理，支持查看证据、做出裁决。

🗄️ 数据库设计

// orders.dispute 字段
dispute: {
  applicant: String,     // 申请人 openid
  reason: String,        // 纠纷原因
  images: Array,         // 证据图片
  status: 'pending'|'resolved',
  result: String,        // 处理结果
  resolveTime: Date
}

☁️ 云函数 API

云函数action功能描述
