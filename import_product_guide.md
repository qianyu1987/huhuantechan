## 导入步骤

### 方法1：云开发控制台导入（推荐）

1. **登录微信开发者工具**
   - 打开你的小程序项目

2. **打开云开发控制台**
   - 点击工具栏 "云开发" 按钮
   - 进入 "数据库" 标签

3. **选择集合**
   - 找到并点击 `products` 集合

4. **导入数据**
   - 点击 "导入" 按钮
   - 选择 `sample_product.json` 文件
   - 确认导入

5. **修改 openid**
   - 导入后找到这条数据
   - 将 `_openid` 字段修改为你的真实 openid
   - 或者删除该字段（系统会自动填充）

### 方法2：使用云函数批量导入

如果你需要批量导入50条数据，我可以帮你：

1. 生成50条特产数据（JSON格式）
2. 编写批量导入云函数
3. 你在控制台执行导入

### 图片处理

当前示例使用占位图片路径 `cloud://placeholder/longjing-tea.jpg`

**替换为真实图片的方法**：

1. **准备图片**
   - 搜索"西湖龙井茶"产品图片
   - 保存到本地

2. **上传到云存储**
   - 云开发控制台 → 存储
   - 上传图片到 `products/` 目录
   - 获取 `cloud://` 格式的 fileID

3. **更新数据**
   - 将 sample_product.json 中的图片路径替换为真实 fileID

## 字段说明

| 字段 | 说明 | 示例值 |
|------|------|--------|
| name | 特产名称 | 西湖龙井茶 |
| description | 详细描述 | 产自杭州西湖... |
| province | 省份代码 | 330000 (浙江) |
| city | 城市代码 | 330100 (杭州) |
| district | 区县代码 | 330106 (西湖区) |
| category | 分类ID | tea (茶叶酒水) |
| valueRange | 价值区间 | 30-50 |
| descTags | 描述标签 | ["绿茶", "明前茶"] |
| images | 图片数组 | ["cloud://..."] |
| daigou | 代购信息 | 价格、库存等 |
| status | 状态 | pending_review (待审核) |
