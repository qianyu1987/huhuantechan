## 首页空白 Bug 修复（20:10）

**问题**：首页完全空白，无内容显示。

**根因**：`pages/index/index.wxml` 里大量使用 `featureFlags.feature_swap`、`featureFlags.tab_publish` 等，但 `pages/index/index.js` 的 `data` 里从未声明或同步 `featureFlags` 字段，导致 WXML 访问 `undefined` 的属性而崩溃。

**修复**：
1. `index.js` data 中新增 `featureFlags: {}`
2. 新增 `_syncFeatureFlags()` 方法，优先从 `app.globalData.featureFlags` 同步，若未就绪则轮询等待（最多3秒）
3. `onLoad` 和 `onShow` 均调用 `_syncFeatureFlags()`

**解决方案二（根治）**：在微信云开发控制台手动创建 `daigouVerify` 集合，设置权限为"仅管理员"。
