## 小程序所有页面无数据问题修复 ✅

### 问题
小程序所有页面无数据显示

### 原因
**HTTP触发参数解析错误** - 所有云函数中多个action使用`event`而非`actualEvent`,导致安卓/鸿蒙端HTTP触发时参数解析失败

### 影响范围
- **productMgr**: list、create、update、detail、remove等所有action
- **orderMgr**: create、list、ship、review、detail、confirm等所有action  
- **adminMgr**: getUsers、getUserDetail、addPoints、deductPoints、editCredit等所有action
- **reviewMgr**: submit、list、myReviews、receivedReviews、checkReviewed等所有action
- **migrateOpenid**: dedupeUsers、removeOpenidField

### 修复内容
✅ 为所有云函数添加HTTP触发参数解析逻辑:
```javascript
let actualEvent = event
if (event.httpMethod && event.body) {
  try {
    actualEvent = typeof event.body === 'string' ? JSON.parse(event.body) : event.body
  } catch (e) {
    return { success: false, error: '请求格式错误' }
  }
}
```

✅ 将所有action中的参数引用从`event`改为`actualEvent`

- ✅ productMgr/index.js

- ✅ adminMgr/index.js
- ✅ reviewMgr/index.js
