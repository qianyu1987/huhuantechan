## 数据验证

### 1. 数据库记录验证

登录云开发控制台，检查以下集合：

**`invite_rewards` 集合**：
```javascript

  "type": "signup" 或 "first_swap",
  "cashAmount": 5.00,  // 现金奖励金额
  "amount": 10,        // 积分奖励数量
  "createTime": "2026-03-30T12:00:00.000Z"
}
```

**`users` 集合**：
- 邀请人用户文档应有 `inviteCode` 字段
- 被邀请人用户文档应有 `invitedBy` 字段

### 2. 现金奖励计算验证

**计算公式**：
- 邀请人总现金奖励 = (邀请人数 × 5) + (首次互换次数 × 10)
- 现金奖励显示 = 总现金奖励（没有奖励显示0）

**示例**：
- 邀请3人，其中2人完成首次互换
- 现金奖励 = (3 × 5) + (2 × 10) = 15 + 20 = 35元
- 页面应显示：¥35.00

## 常见问题排查

### Q1: 现金奖励显示为0
**可能原因**：
1. `system_config` 中没有现金奖励配置
2. `invite_rewards` 集合未创建
3. 云函数未正确部署

**解决方案**：
1. 检查系统配置
2. 创建 `invite_rewards` 集合
3. 重新部署 `userInit` 云函数

### Q2: WXML编译错误
**错误信息**：`WXML 编译错误`
**可能原因**：WXML中使用了 `.toFixed(2)` 方法

**解决方案**：
1. 在JS中预先格式化现金数据
2. WXML中直接显示格式化后的字符串

### Q3: 邀请关系未绑定
**可能原因**：
1. 邀请码无效
2. 防作弊机制阻止
3. 数据库写入失败

**解决方案**：
1. 检查邀请码生成逻辑
2. 查看控制台错误信息
3. 检查数据库权限
