# 互换特产 - 微信小程序

> 全国特产互换平台，发现家乡美食，结交志同道合的朋友。

## 功能特色

### 🏠 核心功能
- **特产发布** - 上传家乡特产，与全国用户互换
- **精准匹配** - 按省份、品类筛选，发现心仪特产
- **互换流程** - 发起→确认→发货→收货→评价
- **神秘特产** - 惊喜盲盒，体验未知乐趣
- **代购特产** - 付费代购想要的家乡味道

### 👥 用户体系
- **信用分机制** - 守信加分，违约扣分
- **省份集章** - 互换成功获得省份印章
- **邀请裂变** - 邀请好友获得积分奖励

### 🎨 设计风格
- iOS 27 液态玻璃设计
- 现代化交互体验
- Emoji 图标增强

## 技术栈

- **前端**：微信小程序 (WXML/WXSS/JavaScript)
- **后端**：微信云开发 (云函数/云数据库/云存储)
- **设计**：iOS 27 液态玻璃风格

## 项目结构

```
├── cloudfunctions/     # 云函数
```

## 开发指南

### 1. 环境准备
- 微信开发者工具
- 微信云开发环境

### 2. 云函数部署
```bash
# 在微信开发者工具中
# 右键 cloudfunctions 文件夹
# 选择"同步云函数列表"
# 右键各云函数选择"上传并部署"
```

### 3. 数据库配置
需要在云开发控制台创建以下集合：
- `users` - 用户表
- `products` - 特产表
- `orders` - 互换订单表
- `daigouOrders` - 代购订单表
- `reviews` - 评价表
- `system_config` - 系统配置表
